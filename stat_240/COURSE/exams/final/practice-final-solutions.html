Problem 16

Over a 167 year period of time, the surface of Lake Mendota is more than 50% covered by ice for \(\bar{y} = 102.0\) days on average with a standard deviation of \(s_y = 19.6\) days. Problem 19

Consider the same setting as the previous problem where the probability that Wisconsin scores a point from a rally in a match against Minnesota is \(p\) and the data is that Wisconsin scored 268 points from 498 rallies. For this problem, we will test the null hypothesis that \(p=0.5\) versus the alternative that \(p\) is something different.

A

Use conventional notation as described in class and write the null and alternative hypotheses.

\(H_0: p = 0.5\)
\(H_a: p \neq 0.5\)

B

A hypothesis test introduced in class uses the observed count of \(X\), the number of points scored by Wisconsin, as the test statistic. For the purposes of this test, condition on the fact that there were a total of \(n=498\) trials this year.

Name the distribution of \(X\) if the null hypothesis is true, providing numerical values for all parameters, and write expressions for how to calculate the mean and standard deviation of this distribution.

Solution

\(X \sim \text{Binomial}(498, 0.5)\)
\(E(X) = 498 * 0.5 = 249\)
\(SD(X) = \sqrt{(0.5)(0.5)/498}\)

C

Write an R expression for how to calculate the p-value for this test

Solution

p_value = 2 * (1 - pbinom(267, 498, 0.5))

D

Assume that the calculated p-value for this test is \(p = 0.097\). Write a summary conclusion about this hypothesis test in the context of the problem, following examples as described in the lectures.

The observed data is consistent with Wisconsin and Minnesota having equal chances to win a point when playing volleyball (\(p=0.097\), two-sided binomial test).

Also okay to say there is very weak evidence that Wisconsin has a better than 50% chance to win a point when playing against Minnesota in volleyball.
